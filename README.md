# Lanchonete do Bairro

Projeto simples de site estático com HTML, CSS e JavaScript.

## Arquivos

- `index.html`: página principal
- `promocoes.html`: página de promoções
- `style.css`: estilos do site
- `cardapio.js`: lógica de exibição do cardápio
- `README.md`: instruções do projeto

## Como usar

1. Abra `index.html` no navegador.
2. Navegue para `Promoções` para acessar a página de ofertas.
